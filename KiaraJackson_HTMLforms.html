Classes

(CMPSC 3323) Internet Programming

Internet programming concentrates on the methodological and technical aspects of software design and programming. it typically focuses on teaching students how to develop web applications using various programming languages, frameworks, and technologies.

(CMPSC 4983) SEM-Cloud Fundamentals

SEM-Cloud Fundamentals is a course that provides an introduction to cloud computing concepts, services, and technologies. It covers fundamental aspects of cloud computing, including the different types of cloud deployment models, essential characteristics of cloud computing, and key cloud service models, Platform as a Service (PaaS), and Software as a Service (SaaS).

(CPSMA 3103) Algorithm Analysis

Algorithm Anaylsis ia an intermediate algorithms course with an emphasis on teaching techniques for the design and analysis of efficient algorithms, emphasizing methods of application. It focuses on evaluating and understanding the efficiency of algorithms. Algorithms are step-by-step procedures for solving computational problems, and analyzing their performance helps determine how well they scale with increasing input sizes.

(CPSMA 2103) Data Structures

Data Structures investigates algorithms for searching and sorting abstract data types, basic algorithm analysis, and recursion. It is also a fundamental component of computer science education, because it focuses on the study and implementation of various algorithms and data structures for managing, organizing, and storing managing data efficiently.
